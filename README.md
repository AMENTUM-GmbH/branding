## 🛠 Verwendung

Diese Assets sind für die Verwendung in internen Projekten, Web-Interfaces und offiziellen Dokumentationen vorgesehen. 

Bei der Einbindung über Docker oder Web-Dienste sollte bevorzugt der **Raw-Link** verwendet werden, um eine korrekte Darstellung zu gewährleisten.

### Beispiel für die Einbindung (Umgebungsvariable):
`WEBUI_NAME="AMENTUM"`
`WEBUI_FAVICON_URL="https://raw.githubusercontent.com/AMENTUM-GmbH/branding/main/logos/amentum-logo.png"`
